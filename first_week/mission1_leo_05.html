미션1. 빈칸 채우기(1)

HTML은 HyperText Markup Language의 약자로 웹을 이루는 가장 기초적인 구성 요소입니다. 웹페이지의 모양을 꾸미는데 쓰는 CSS와 기능, 동작을 구현하는데 쓰는 JavaScript와 함께 쓰입니다.
HTML 요소는 태그를 사용해서 다른 텍스트와 구분합니다. 태그는 여는 홀화살괄호(<>)로 감쌉니다.
프론트엔드 개발자는 HTML, CSS, JavaScript를 사용해 웹을 만듭니다.

미션2. 빈칸 채우기(2)

HTML 태그엔 속성을 추가할 수 있습니다. 속성은 태그에 추가 정보를 제공하거나 태그의 동작이나 표현을 제어할 수 있게 해줍니다.
속성은 이름과 값으로 이루어져 있습니다. 속성값은 홑따옴표(')와 겹따옴표(")로 감싸 표현합니다.
한 태그에 여러 속성을 선언할 수 있는데, 여러 속성을 선언할 때는 공백으로 구분해서 사용합니다.
(참조: HTML 요소 분석 - HTML 기본 - Web 개발 학습하기 )

미션4. 요소별 사용된 태그 확인

  1. 위쪽의 NAVER 로고: a, h1
    • a 태그를 사용하여, 클릭시 네이버 메인으로 이동할 수 있게 함.
    • 로고 이미지 경로의 경우 CSS를 활용하여 a태그에 적용하였음.
    • a 태그 하위에 h1 태그로 'NAVER'를 적어두었음(접근성).
    • h1 텍스트의 경우 CSS에서 'clip: rect(0 0 0 0)'을 걸어놔서 보이지않게 설정하였음.

  2. 아이디와 비밀번호를 입력하는 네모 상자: div, span, input,
    • 'id_pw_wrap'(div)로 묶고 아래에 각각 'input_row'(div) 란 이름으로 묶었음.
    • 그 아래에 'icon_cell'(span), 'input_text'(input), 'button'(span)을 나란히 배치함.
    • 'icon_cell'의 경우, 각 아이콘의 이미지를 불러오는 'icon_{name}'(span)과 접근성을 위한 'blind'(span)를 설정함.
    • 'input_text'의 경우, placeholder 속성을 통해 입력 전에만 각각 '아이디'와 '비밀번호'란 글자를 표시하게 함.
    • 'button'의 경우, 한 글자라도 입력하였을 시 우측에 Ⓧ표시가 나오게 하여('icon_delete'), 누르면 내용이 지워짐.
    • 'button'의 하위에도 'icon_delete'외에 접근성을 위한 'blind'(span)를 설정함.

  3. 초록색 로그인 버튼: div, button, span
    • 'btn_login_wrap'(div) 아래 'submit'이라는 타입 속성을 적용한 'btn_login'(button)이 있음.
    • 그 하위에 'btn_text'(span)이 있고 버튼에 쓰여진 글자 '로그인'을 가지고 있음.
